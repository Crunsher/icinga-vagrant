Welcome to the Icinga 1.x Demo Vagrant Box!

This demo vm is installed with the help of Vagrant, Puppet and Icinga packages from the snapshot repository located at http://packages.icinga.org. Base system is a CentOS 6.x i386 virtualbox image.

If you're looking for the Icinga 2 Vagrant Box, please checkout https://www.icinga.org/icinga2/download/

SSH

SSH is available through 'vagrant ssh' with passwordless sudo ('sudo -i').

Databases

The puppet modules will take care of installing the required databases. The packages themselves do not install/upgrade any database schema. Please consult the package provided README.RHEL* files for local upgrades (e.g. /usr/share/doc/icinga-idoutils-libdbi-mysql-$version/README.RHEL.idoutils).
